Slide show

De slider maakt het mogelijk om meerdere afbeeldingen te laten zien zonder dat daar meer ruimte voor nodig is. De slide kan zowel handmatig als tegelijkertijd met een timer worden doorgeschoven. De code die gebruikt wordt maakt het ook mogelijk meerdere slides op een pagina te gebruiken.

De code

De code bestaat uit een drietal functies. De bovenste functie autoSlide regeld het automatische doorschuiven van de de slides. De tweede functie plusSlides wordt gebruikt om de volgende slider te beinvloeden als er meedere sliders op de pagina aanwezig zijn. De laatste functie showSlides regeld de uiteindelijke slide.
