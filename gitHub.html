GitHub Desktop

GitHub Desktop is een applicatie voor github om git tree's te maken. De applicatie kan verboden worden met een GitHub account zodat je code projecten kan sychoniseren met "GitHub". Deze applicatie is grotendeels gratis en gericht op de basis van git trees.

Wat is Git

De website van git zegt het volgende: "Git is a free and open source distributed version control system designed to handle everything from small to very large projects with speed and efficiency.". Simpel gezegt is git een control systeem voor software development. Git wordt gebruikt voor versie beheer doormiddel van een "tree" en "branches".
Zodoend helpt git programeur bij het developen van programma's en het beheren van verschillende versies en varianten hiervan.

Tree en Branches

Wanneer je een nieuw project begint en je maakt een nieuwe git directory aanmaakt begin je met de "Tree". De Tree kun je zien als de meeste complete versie van het programma. Je kan meervarianten van hetzelfde programma archieveren door een "branch" aan te maken. Een branch is een aftakking van je tree ookwel main-branch genoemd. Deze branch loopt parallel aan je tree. Bezoek de website van Atlassian voor een gedailtalleerde uitleg over git.

GitHub

GitHub is een online platform voor git directorys. GitHub wordt gebruikt door developers om hun git naar de cloud te halen. Daarbij komt dat als je met meerdere developers aan hetzelfde project werkt er automatisch een centraal punt ontstaat waar je met elkaar kan samenwerken.

GitHub Desktop

Met behulp van GitHub Desktop kan je projecten naar GitHub pushen. Met 1 druk op de knop. Voordat je begin koppel je een projectmap op je computer aan een directory op GitHub. GitHub Desktop registreerd de wijzigingen en kijkt geregeld of er wijzigingen hebben plaats gevonden om GitHub.

GitHub helpt je op een overzichtelijke manier weer te geven wat je hebt aangepast/wat er aangepast is op GitHub.
