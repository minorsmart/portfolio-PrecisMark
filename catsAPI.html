Cat Facts API

De cat facts API genereerd verschillende feiten over katten. De api call vind plaats bij het laden van de pagina en wanneer een nieuw verzoek wordt gevraagd.
Wanneer een API call wordt gestart wordt de volledig data-set opgevraagd. De data wordt omgezet in een .json format. Tot slot wordt een "random" nummer gerenegeerd die wordt gebruikt om de string te vinden en weer te geven in een html element.
