BCP

Een Printed Curcuit Board ookwel kort een BCP genoemd. Is bord waar elektronische componeneten op gesoldeerd worden. Een PCB wordt gebruikt om een elektronisch curcuit op te maken. Om elektronische componeneten met elkaar te verbinden.

Op de vorige pagina was dit curcuit op een breadboard ontworpen. Dit heeft als voordeel dat je de componeneten gemakkelijken kan monteren en demonteren.

Een BCP ontwerpen

Voordat een BCP "geprint" kan worden moet deze eerst ontworpen worden. Dit kan doormiddel van een tal van verschillende applicaties. In de workshop "Smart Technologie" werd Autodesk Eagle gebruikt. Maar ook Fritsing is een vergelijkbaar programma waarmee BCP's ontworpen kunnen worden.

PCB Schema ontwerpen

Voordat je het curcuit gaat ontwerpen pak je eerst alle benodigde onderdelen. Vervolgen label je alle in- en uitgangen. Het maakt niet uit op waar je de onderdelen plaat op het scherm. Het enige waar je op hoeft te letten is de juiste labeling en de juiste componenten.

PCB Layout ontwerpen

Vervolgen kun je het schema omzetten in een layout. In de layout editor ga je de afmetingen van de PCB bepalen. Ook plaats je hier de componenten op de plek waar je die wilt hebben.
Omdat je in de schema editor de in- en uitgangen met behulp van zelfde labeling aan elkaar hebt gekopeld. Kan in de layout editor met een druk op de knop de kanalen worden gelegd. Vaak gaat dit de eerste keer vrij goed. Soms moeten er nog handmatige aanpassingen worden gedaan.
