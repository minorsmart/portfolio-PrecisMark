Smart Technologie

Bij "Smart Technologie" maak je kennis met verschillende technologiën. In het eerste deel wordt er gewerkt aan het programmeren en maken van een weerstation dormmiddel van een esp32 controler. Nadat het ontwerpproces is afgerond wordt er een PCB (Printed Curcuit Board) ontworpen.

Kennismaken met de componenten

In het begin van het ontwerp proces ga je de verschillende componenten uittesten. Voor het weerstation heb je de volgende componenten nodig:

  • ESP32 microcontroller (Espressif Systems Processor)

  • LED en bijhoren weerstanden (~220 ohm)

  • LDR sensor (Light Depended Resistor

  • SHS (Soil Humidity Sensor)

  • DHT sensor (a humidity and temperature sensor

  • Relay

from machine import Pin
import utime

p2 = Pin(21, Pin.OUT)    # create output pin on GPIO2

while True:
    p2.on()              # set pin to "on" (high) level
    utime.sleep_ms(1000) # small delay
    p2.off()
